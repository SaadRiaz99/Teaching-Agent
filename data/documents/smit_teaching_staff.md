# SMIT Teaching Staff Information

## Teaching Staff

### Madam Sumaiya
- **Role:** AI & Data Science Instructor
- **About:** Madam Sumaiya teaches Artificial Intelligence, Machine Learning, Python Programming, and Data Analysis. She helps students understand AI concepts through practical projects and hands-on exercises.
- **Subjects:** Python Programming, Machine Learning, Data Analysis, Generative AI, Prompt Engineering
- **Teaching Style:** Project-based learning. Encourages students to solve real-world problems using AI technologies.

### Sir Talal
- **Role:** Web & Software Development Instructor
- **About:** Sir Talal teaches modern web development technologies and software engineering principles. He guides students in building full-stack applications.
- **Subjects:** HTML, CSS, JavaScript, React, Node.js, Database Management
- **Teaching Style:** Emphasizes clean code, software architecture, and practical development skills.

### Saad Bin Riaz
- **Role:** Cloud & DevOps Instructor
- **About:** Saad Bin Riaz teaches cloud computing, DevOps practices, and deployment strategies. He helps students understand modern infrastructure and automation tools.
- **Subjects:** Cloud Computing, Docker, Kubernetes, CI/CD, Linux Administration, AWS Fundamentals
- **Teaching Style:** Focuses on industry-standard tools and real-world deployment scenarios.

4. Students are encouraged to collaborate and participate in discussions.
5. AI tools should be used responsibly and ethically.
